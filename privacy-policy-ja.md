# Trakio プライバシーポリシー

最終更新日: 2026年4月5日

## 概要

Trakio はローカルファーストで設計されています。共有機能や外部連携、または直接のお問い合わせを利用しない限り、ほとんどのセッションデータは端末内に保存されます。

## 端末に保存されるデータ

Trakio は以下のデータを端末内にローカル保存します。

- トラックデータ
- セッション
- ラップ
- セクタータイム
- GPS サンプル
- セッションノート
- 車両情報
- セッションに保存された天候情報
- ローカルのドライバープロフィール

ローカルプロフィールには以下が含まれる場合があります。

- ユーザー名
- 車両名
- 国籍 / 国コード
- アバター画像
- 任意のリーダーボード共有で使用するローカル publisher ID

## 利用する権限

### 位置情報

Trakio は、トラック走行中の GPS テレメトリー記録、およびラップ・セクター通過判定のために位置情報の権限を使用します。

### フォトライブラリ

Trakio は、以下の場合にのみフォトライブラリへのアクセスを使用します。

- アバター画像を選ぶ場合
- ストーリー共有用の写真を選ぶ場合
- 生成した画像をフォトライブラリへ保存する場合

これらの権限を許可しない場合、該当機能は利用できません。

## ネットワークサービス

Trakio は以下のサービスにネットワーク経由で接続することがあります。

- **Open-Meteo**: 選択中トラックの現在天気取得
- **ホストされたリーダーボードサービス**: ラップタイム共有時
- **Instagram**: セッションを Instagram ストーリーズへ共有する時
- **端末のメールアプリ**: フィードバック送信時

## リーダーボード共有

リーダーボード共有は任意です。

ラップを共有する場合、Trakio は以下のデータをホストされたリーダーボードサービスへ送信します。

- track ID
- 共有ユーザー名
- 国コード
- 車両名
- ラップタイム
- 共有日時
- 同一トラックで後からタイム更新できるようにするための安定したローカル publisher ID

共有されたリーダーボード情報は、他の利用者に表示されることを前提としています。

## データ保持と削除

ローカルのセッション、ノート、プロフィール情報、およびキャッシュされたアプリデータは、利用者が編集・削除するか、アプリを削除するまで端末内に残ります。

共有したリーダーボード情報は、後で関連するローカルセッションを端末から削除しても、ホストされたリーダーボード上に残る場合があります。

現在、Trakio にはホストされたリーダーボード情報をアプリ内から削除するセルフサービス機能はありません。

## サードパーティ解析、広告、トラッキング

現在の Trakio は以下を使用していません。

- サードパーティ広告 SDK
- 広告目的のユーザートラッキング
- 必須のアカウントログイン
